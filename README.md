# ZERA

Site vitrine officiel du projet **ZERA**, une serie audio sombre et immersive centree sur la peur, le controle et la survie.

## Apercu

Le site presente :

- l'univers de `ZERA`
- la vision du projet
- le recrutement et le casting
- les moyens de contact

## Site en ligne

Le site est publie via GitHub Pages.

- URL : `https://floflif06.github.io/ZERA/`

## Fichiers principaux

- `index.html`
- `style.css`

## Mise a jour du site

Pour modifier le site :

1. mettre a jour `index.html` ou `style.css`
2. envoyer les fichiers sur le depot GitHub
3. attendre quelques minutes que GitHub Pages republie le site
